# translator

- Scala 2.9 with Play2 and Salat
- Backbone
- MongoDB 2.2

# Installation

```

# Requirements
mongodb

# Install play 2.0.4
git clone https://github.com/playframework/Play20.git
cd Play20
git checkout 2.0.4
cd framework
./build
> publish-local
export PATH=$PATH:/relativePath/to/play

# Run translator
git clone https://github.com/dennis84/translator.git
cd translator
play
> run 8000

```
